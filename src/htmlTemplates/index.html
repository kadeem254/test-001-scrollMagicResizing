Basic webpack template

This is a basic web template meant to ease creation of new projects that require webpack, in the form i find myself generating most of the times.

Once cloned the needed libraries should easily be set by use of the command. npm install That will generate the node modules required. To easily test out if the template is functioning the 'build' command specified in the 'package.json' file can be run using the command. npm run build
